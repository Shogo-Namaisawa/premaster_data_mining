iris¶

iris データセットにはアヤメの花を4つの特徴量（'sepal length (cm)', 'sepal width (cm)', 'petal length (cm)', 'petal width (cm)'）で表した150個の特徴量ベクトルが含まれています(iris['data'])。

また、各花には3種類（0:'setosa', 1:'versicolor', 2:'virginica'）のいずれかがラベルとして付与されています（ iris['target'] )

scikit-learnでの機械学習手順¶

この色ごとにクラスタリングしていくよ〜〜〜〜

  • 使用するモデルのクラスの選択

  • モデルのハイパーパラメータの選択とインスタンス化

  • データの準備

    • 教師なし学習では、特徴量データを準備

  • モデルをデータに適合（ fit() メソッド）

  • モデルの評価

    • 教師なし学習では、 transform() または predict() メソッドを用いて特徴量データのクラスタリングや次元削減などを行う
